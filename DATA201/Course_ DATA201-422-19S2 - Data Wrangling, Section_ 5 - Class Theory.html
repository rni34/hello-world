5 - Class Theory

  • 5 - Class Theory

    5 - Class Theory

    What kind of stuff is that?

    Your progress

    • Lab 5

      Variables in R can be of different types. Not taking into account this diversity is bad bad not good.

      In this lab we will see how to identify and deal with different class of objects (dates, numerics, strings of characters, more exotic stuff).
    • Lab 5 Material Folder
    • Reading for lab 5 Julia: Types URL

      Data types in Julia are very rich and deep. Julia is somewhat closer to an object oriented language, and a lot of Julia functionalities come from properly defining data types and structure. Read through the latest version of Julia manual to understand it more.

    • Reading for Lab 5 Julia: "vectors" URL

      Some of the additional functionalities that R receives from purrr are available natively in Julia. See this man page (after the one about types) to get an overview.

    • Lecture 6

      Before we review and discuss what we have seen so far.

      Then we will talk a tad about bringing data from the web to our universe.

      No lecture notes for this lesson, as it was mostly based on discussion with the class. Instead, please refer to two reading from Hadley's Wickham's Advanced R:
      • https://adv-r.hadley.nz/vectors-chap.html
      • http://adv-r.had.co.nz/Data-structures.html
